# 🚀 Space Shooter

A modern HTML5 Space Shooter game built using **HTML, CSS, and JavaScript** with no external libraries. Battle enemy ships, defeat bosses, collect power-ups, and survive as long as possible in an endless space adventure.

## 🎮 Features

- 🚀 Smooth spaceship controls
- 👾 Multiple enemy types
- 💀 Boss battles
- 🔫 Rapid Fire power-up
- 💣 Screen-clearing Bomb power-up
- ✨ Particle explosion effects
- 🌌 Animated starfield & nebula background
- 🔊 Sound effects & background music
- ❤️ Life system
- 📈 Level progression
- 🏆 Best score tracking
- 📱 Mobile touch controls
- 💻 Desktop keyboard controls
- 🎨 Modern animated dashboard/menu

---

## 🎮 Controls

### Desktop

| Key | Action |
|------|--------|
| ⬅️ / ➡️ | Move spaceship |
| Space | Shoot |
| R | Restart game |
| Esc | Return to Home Menu |

### Mobile

- 🎮 Drag the joystick to move
- 🔫 Hold Fire button to shoot
- 💣 Tap Bomb button to destroy all enemies

---

## 🕹 Gameplay

- Destroy enemy ships to earn points.
- Every **100 points** increases your level.
- Enemies become faster as levels increase.
- Boss enemies appear periodically.
- Collect power-ups to gain special abilities.
- Survive as long as possible and beat your high score!

---

## ⚡ Power-Ups

### 🔫 Rapid Fire

- Doubles firing speed
- Lasts for 10 seconds

### 💣 Bomb

- Instantly destroys every enemy on screen
- Awards bonus points
